# TailTimes

TailTimes is a mobile-first app that helps pet sitters share updates with pet owners during boarding sessions.

## Core Idea

Instead of sending scattered updates via WhatsApp, TailTimes provides:

- A private session feed (via link, no login)
- A structured timeline of updates
- A portfolio built automatically from past sessions

## Goals

- Reduce friction for sitters when sharing updates
- Provide better visibility and reassurance for pet owners
- Help sitters build trust and a professional image

## MVP Features

- Create sitter profile
- Create session
- Upload photo/video
- Share session link
- View updates (owner side)

## Tech Stack (tentative)

- Next.js (App Router)
- Tailwind CSS
- Vercel (hosting)
- Clerk (auth)
- Drizzle (orm)
- Supabase using Postgresql(DB + storage)
