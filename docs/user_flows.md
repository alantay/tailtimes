# User Flows

## Sitter Flow (Primary)

1. Create session
2. Share link with owner
3. Open app
4. Take photo
5. Upload instantly
6. Repeat

Goal: < 3 seconds per upload

---

## Owner Flow

1. Click link
2. View timeline
3. Scroll updates
4. (Optional) revisit later

No login required

---

## Portfolio Flow (Future)

1. Session ends
2. Sitter selects highlights
3. Auto-generated page added to profile
