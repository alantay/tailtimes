# MVP Scope

## In Scope

- Sitter profile (basic)
- Session creation
- Media upload (image/video)
- Session feed page (public via link)
- Simple UI

## Out of Scope

- Notifications
- Reviews
- Messaging
- Payments
- Multi-user accounts

## Success Criteria

- Sitter uploads at least 3 updates per session
- Sitter returns to use TailTimes again
